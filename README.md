# 키움증권 OpenAPI+ 기반 자동 트레이딩 봇  
실시간 데이터 수신 및 조건 검색을 진행하고, 설정된 조건에 따라 자동으로 매매를 진행하는 프로그램 입니다

## 📦 개발 환경

- IDE: Visual Studio Code 2019
- Framework: .NET Framework 4.7.2
- Language: C#
- API: Kiwoom OpenAPI+
- Platform: Windows 64bit

---

## 🚀 주요 기능

- 키움 OpenAPI+ 로그인 및 연결 상태 체크
- 실시간 시세 수신 및 분석
- 조건 검색식 자동 조회
- 매수/매도 조건 트리거 설정
- 거래 로그 저장 및 시각화
